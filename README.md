# radish-legal

Public legal pages for the **Radish** iOS app (privacy policy), served via GitHub Pages.

- Live: https://coleocofer.github.io/radish-legal/
- Source of truth for the policy text also lives in the app repo at `docs/privacy-policy.md`.

This repo is intentionally **public** (the app's own source repo is private) and contains only the
hosted legal text — no app code.
